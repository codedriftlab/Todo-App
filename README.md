# Todo-App
A Todo App is a simple web application used to manage daily tasks. Users can add new tasks, view the list of tasks, and delete tasks when they are completed. This helps users stay organized and improve productivity.

How It Works (GitHub Flow)
GitHub stores project code online 👉 GitHub
git clone → downloads code to your system
Goto directory → Go inside the file
npm install → installs libraries
npm start → runs the app
